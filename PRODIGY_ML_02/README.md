# Customer Segmentation using K-Means Clustering

## Overview

This project applies K-Means Clustering to segment customers of a retail store based on their annual income and spending score.

## Dataset

Customer Segmentation Dataset

Source:
https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python

File Used:
- Mall_Customers.csv

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Features Used

- Annual Income (k$)
- Spending Score (1-100)

## Workflow

1. Load dataset
2. Perform exploratory analysis
3. Determine optimal clusters using Elbow Method
4. Train K-Means model
5. Visualize customer segments

## Results

Customers are grouped into distinct clusters based on purchasing behavior, helping businesses identify different customer categories for targeted marketing.

## Conclusion

K-Means Clustering successfully segmented customers into meaningful groups based on annual income and spending patterns.
